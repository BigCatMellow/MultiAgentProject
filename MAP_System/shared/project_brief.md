# Project Brief

## Objective

Create a general-purpose multi-agent project workspace where Codex and Claude Code can collaborate through structured tasks, shared durable context, handoffs, reviews, and a future LangGraph runtime.

## Completion Condition

The workspace is ready when:

- both agents have clear shared operating instructions;
- project truth, requirements, decisions, and unresolved questions have canonical locations;
- tasks are represented as structured files;
- a task graph can be mechanically validated;
- LangGraph has a clear local scaffold and integration notes;
- future project work can be added without relying on unstructured chat history.

## Current Baseline

The source architecture is `general_purpose_multi_agent_project_system_v2.md`.

The stress-test note `Story_logic_test.md` is preserved as a project artifact/source note and should not be treated as a runtime protocol document.
