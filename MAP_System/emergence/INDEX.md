## ID format

- Insights: `INS-0001`, `INS-0002`, ...
- Synthesis: `SYN-0001`, `SYN-0002`, ...

**Experiment:** PROPOSED -> APPROVED -> RUNNING -> SUBMITTED -> REVIEWED -> ADOPTED / REJECTED / ARCHIVED
