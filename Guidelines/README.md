# Guidelines

Universal AI collaboration protocols for this workspace. These apply to **any
project** here — MAP, Pathwell, or anything added later. They are not
MAP-specific and contain no task boards or project state.

Read a file here when you need to answer a protocol question that is independent
of which project you are working on.

For MAP-specific operating rules, see `MAP_System/AGENTS.md` and `MAP_System/notes/`.

| `phatic-suppression.md` | How to eliminate low-value filler in agent messages |
| `laziness-ladder.md` | How to pick the cheapest capable agent for a task |
