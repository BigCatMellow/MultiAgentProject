# MultiAgentProject

Project files are kept at the top level. The multi-agent coordination system lives in:

```text
MAP_System/
```

Useful entrypoints:

- `MAP_System/README.md` - MAP system overview
- `MAP_System/AGENTS.md` - shared Codex/Claude rules
- `MAP_System/CLAUDE.md` - Claude Code instructions
- `MAP_System/scripts/map-git` - Git wrapper for this workspace

Git status:

```bash
MAP_System/scripts/map-git status
```
