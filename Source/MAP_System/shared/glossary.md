# Glossary

**Agent:** A model-backed worker such as Codex or Claude Code operating with task-specific context.

**Artifact:** A durable work product stored in the repository.

**Handoff:** A structured transfer note from one agent to another.

**LangGraph:** A workflow orchestration library for stateful graph execution, branching, loops, checkpoints, and human pauses.

**Task Graph:** The dependency graph that says which tasks are ready, blocked, in review, or complete.

**Task Owner:** The one agent accountable for a task's primary output while it is active.
