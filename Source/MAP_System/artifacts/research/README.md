Staging area for research artifacts. Empty until tasks produce research outputs.

See `artifacts/README.md` for the full artifact taxonomy.

This folder is intentionally empty by decision, not by neglect: see
`shared/decisions.md` DEC-027. The Research System is fully built
(`RESEARCH_SYSTEM.md`, `templates/research/`, `validate_research_artifacts.py`)
but no task has yet had a genuine external/contested-fact research need. The
first task that does must use the templates here rather than skip them.
