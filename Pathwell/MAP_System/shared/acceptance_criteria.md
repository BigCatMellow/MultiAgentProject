# Pathwell MAP Acceptance Criteria

- `AGENTS.md` defines shared collaboration rules.
- `CLAUDE.md` gives Claude Code a direct local entrypoint.
- `workflow/task_graph.json` exists and validates.
- `tasks/` contains initial actionable task files.
- `langgraph/` contains a starter orchestration module and explanatory notes.
- `scripts/validate_task_graph.py` can validate task graph consistency.
- `shared/` contains project brief, requirements, constraints, decisions, unresolved questions, glossary, and agent rules.
- Initial Pathwell tasks reference the story files in `../Story_Files/`.
