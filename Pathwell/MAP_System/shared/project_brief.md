# Pathwell Project Brief

## Objective

Use the MAP collaboration system to develop and review the Pathwell story project with Codex and Claude Code working through structured tasks, durable notes, handoffs, and independent review.

## Completion Condition

The project is ready for its next milestone when:

- the current story canon and notes have been inventoried;
- major therefore/but chain gaps are captured as actionable tasks;
- revisions are made against explicit acceptance criteria;
- review notes distinguish required fixes from optional taste notes;
- final outputs live in `Story_Files/`, `MAP_System/artifacts/`, or another declared path.

## Current Baseline

Primary story files live in `Pathwell/Story_Files/`:

- `canon.md`
- `character_bible.md`
- `world_bible.md`
- `forbidden_patterns.md`
- `glossary.md`
- `Story_logic_test.md`
- `Pathwell_convenience_paradox.md`

The root MAP system remains available at `../MAP_System/`, but this Pathwell copy should be treated as the project-local coordination system.
